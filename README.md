# Environmental-Data-Analysis
The aim of this project is to perform analysis on data gathered on toxins in a river for a non-profit body to uncover insights that can drive conservation efforts and policy recommendations. 
To achieve this, the data is imported, cleaned and then basic analyses conducted on the data set. Then visualisations are created to communicate the findings. 
The dataset used is the “National River Toxin Dataset”, which contains information about various toxin levels, pH levels, temperature, and geographical data for multiple over systems over the past few years. 

Tools used are Python 3.13.12, Anaconda Navigator, Jupyter Notebook, and Python Libraries: pandas, numpy, matplotlib, seaborn, scipy
